iSticky

iPad [and iPhone:orientation] sticky footer template

To show how it's possible to target devices using css media queries though due to uneven browser support additional client-side detection will deliver a better user experience across all browsers and platforms.

The relevant media queries without src attributes:

<link media="handheld, only screen and (max-width: 480px)"/> <link media="screen and (min-width: 481px)"/> <link media="screen and (min-width: 768px) and (max-width: 1024px)"/> <!--[if lt IE 9]> <link rel="stylesheet" type="text/css" href="ie.css" media="all"/> <![endif]-->

The relevant sticky footer CSS commented in styles-ipad.css

The relevant sticky footer html tag structure: [thanks to Ryan Fait]

<body> <div class="wrapper"> Content <div class="push"></div> </div> <div id="footer"></div> </body>
